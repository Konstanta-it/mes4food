# Показатели анализов номенклатуры

Справочник предназначен для хранения показателей которыми в дальнейшем можно будет воспользоваться при создании [вида](TypesOfAnalyzes.md).

**"Показатели анализов номенклатуры"** располагается в подсистеме **"Производство"** в разделе **"Управление качеством"**:

[![1][1]][1]

При создании показателя необходимо заполнить: 

- Наименование
- Вид результата анализа

[![2][2]][2]

Опционально можно прикрепить файл относящийся к показателю:

[![3][3]][3]

При нажатии на **"Файл"** откроется новое окно, в котором можно выбрать файл.

[![4][4]][4]
